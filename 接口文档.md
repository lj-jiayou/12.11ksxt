```

```

#### 接口描述
```
注册用户
```
#### 接口地址

```
http://localhost:8080/api/registry
```

#### 接口请求方式
```
POST
```

#### 接口参数
```
参数名称| 参数说明
---|---
 username| 用户名
 password| 密码
 usercode| 学号(可选)
 identity| 身份id
```
#### 接口返回值
```
{code: 1, msg: "注册成功"}
code: 1
msg: "注册成功"

```
